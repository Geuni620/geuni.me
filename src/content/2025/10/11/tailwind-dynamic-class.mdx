Tailwind CSS(이하 Tailwind)는 동적으로 클래스를 생성하지 않는 것으로 알고 있다.  
즉, 다음과 같은 코드는 작동하지 않는다.

```tsx
const Parent = () => {
  return (
    <div>
      <div className={`w-[${width}px] h-[${height}px]`}>Child</div>
    </div>
  );
};
```
